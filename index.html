CoffeeScript

Syntactic sugar never tasted so sweet!

It's just JavaScript

CoffeeScript is an attempt to expose the good parts of JavaScript in a simple way.
  • Less clutter
  • More concise
  • Everything is an expression

Functions

  "Filling the #{container}
    with #{liquid}."
  

product = function(x, y) {
  return x * y;

msg = function(container, liquid) {
  if (liquid == null) {
    liquid = "coffee";
  }
  return "Filling the " + container +
    "with " + liquid + ".";
};

Documentation goes here and is not indented. You can write as much as you
want. Just change the file extension to .litcoffee so the compiler knows
how to read it. It will treat the document as a markdown document. Code
is indented (like normal markdown).
