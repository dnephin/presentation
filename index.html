It's just JavaScript

CoffeeScript is an attempt to expose the good parts of JavaScript in a simple way.

msg = function(container, liquid) {
  if (liquid == null) {
    liquid = "coffee";
  }
  return "Filling the " + container +
    "with " + liquid + ".";
};
